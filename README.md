### BridgeLabz

#### 10 Dec, 2024

**Commands:**

- **Git Merge**:  
  a. `git merge branch`  

- **Git Clone**:  
  a. `git clone <repo-url>`
  b. `git clone -b <branch-name> <repo-url>`  

- **Git Pull**:  
  a. `git pull`  
  b. `git pull origin <branch-name>`  

- **Git Fetch**:  
  a. `git fetch`  
  b. `git fetch origin <branch-name>`  

- **Git Stash**:  
  a. `git stash`
  b. `git stash push -m "Mesaage"` 
  c. `git stash list`
  d. `git stash pop`
